## How to Use This File

When preparing a new release:
1. Move items from "Unreleased" to a new version section
2. Add the version number and release date
3. Group changes under "Added", "Changed", "Deprecated", "Removed", "Fixed", or "Security"
4. Keep the "Unreleased" section for ongoing development

Example:
```
## [0.1.0] - 2024-12-28

### Added
- Initial release with core gameplay features
```
